About the Project

The Intelligent Traffic Control System (iNovaTCS) is an AI-powered solution designed to transform urban mobility. It uses real-time video surveillance, computer vision, and deep learning models to detect traffic patterns and anomalies, including accidents, fires, and congestion. The system dynamically adjusts traffic signals to optimize flow, minimize delays, and enhance emergency response. Through a centralized web-based dashboard, authorities can monitor live footage, receive alerts, view incident reports, and analyze historical traffic data — all aimed at building safer, smarter, and more responsive cities.

  • Real-time vehicle detection and object tracking using AI models (YOLOv8)
  • Automatic detection of anomalies such as accidents and fires
  • Dynamic traffic signal control based on live vehicle count and flow
  • Centralized dashboard for monitoring, alerts, and incident reporting
  • Historical traffic and anomaly data for insights and analytics
  • Improved emergency response with instant notifications and evidence capture

Project Specifications

  • Real-time traffic monitoring using CCTV video feeds
  • AI-based anomaly detection including accidents, fires, and traffic congestion (YOLOv8)
  • Automated generation of incident reports with visual evidence and timestamps
  • Interactive and responsive web dashboard built with Django for live control and analytics
  • PostgreSQL database integration for storing traffic data, reports, and system logs
  • Modular architecture for easy integration with emergency services and city infrastructure
